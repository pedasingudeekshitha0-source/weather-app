# weather App 
A simple weather App built using HTML, CSS, and JavaScript that provides real-time weather information based on city search.

---

## 🚀 Features

- Search weather by city name
- Real-time temperature display
- Weather condition updates
- Responsive user interface
- Beginner-friendly project
- Fast and lightweight application

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
